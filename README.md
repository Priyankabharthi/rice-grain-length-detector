# Rice Grain Length Detector

This project is a Rice Grain Length Detection system developed using Python, OpenCV, and Image Processing techniques.

## Features
- Detects rice grain length automatically
- Image processing using OpenCV
- Grain size analysis
- Simple user interface
- Fast and accurate detection

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## Project Objective
The objective of this project is to analyze rice grain images and calculate grain dimensions for quality analysis.
